# SHOP-ME-DJANGO-PROJECT<br>
Author - Sonu Thakur <br>

Mini E-commerce Store ( Django + MySQL ) 
• Implemented a secure User Authentication and Cart System with features like login, signup, session handling, 
and real-time cart updates to enhance user experience and security.<br> 
• Developed a Dynamic Product Management module, allowing admin users to add, edit, delete, and 
manage products using Django ORM with fully automated database operations. <br>
• Designed the entire website to be fully responsive, ensuring smooth performance and consistent UI across 
mobile, tablet, and desktop devices using Bootstrap and custom CSS. <br>
• A fully functional live demo of this project is available at [https://sonu99.pythonanywhere.com/] showcasing the 
complete authentication flow, product management system, and responsive design.
